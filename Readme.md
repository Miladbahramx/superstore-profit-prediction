# 📊 Superstore Profit Prediction

## 📌 Project Overview

This project analyzes retail sales data and builds Machine Learning models to predict profit based on sales transactions.

---

## 🎯 Business Problem

Can we predict the profit of an order using information such as:

- Sales
- Discount
- Quantity
- Category
- Region

The goal is to understand which factors have the biggest impact on profit.

---

## 📂 Dataset Information

Dataset: Superstore Sales Dataset

Rows: 10,194

Columns: 21

Main Features:

- Sales
- Profit
- Quantity
- Discount
- Category
- Region
- Order Date

## 🤖 Machine Learning Models

### Linear Regression

R² Score: 0.41

Mean Absolute Error: 61.32

---

### Random Forest Regressor

R² Score: 0.74

Cross Validation Mean R²: 0.76

---

## 🏆 Key Insights

- Technology category generates the highest profit.
- West region has the highest sales.
- Sales is the most important feature.
- Discount has a significant impact on profit.
- Random Forest significantly outperformed Linear Regression.
